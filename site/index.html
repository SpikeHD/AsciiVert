            AsciiVert was originally created as something to work on over weekends. It ended up being something I've worked on
            pretty regularily in my free time.
            


            If you're interested in technical details, this whole site runs on through Nginx on a cheap $5 DigitalOcean droplet.
            The processor AND webserver are run using NodeJS. The routes are handled with Express and the processor itself uses
            an orchestra made up of Canvas, Jimp, and FFMPEG. There is no doubt that it could be better by using a lower level
            language and optimizing it profusely, but it is what it is.
            


            I guess one warning I have, if you're visiting and planning on trying it, is that because of the small, limited server
            it runs on, it frequently runs out of memory and stops processing your video/image. The best way to minimize the chance
            of this happening is just making sure your resolution and framerate are low.
